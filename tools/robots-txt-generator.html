How to Use
  1. Select default access (allow all or disallow all)
  2. Enter your sitemap URL (optional)
  3. List any paths to disallow
  4. Click 'Generate Robots.txt'
  5. Download and upload to your website root

            
Features
  • Allow or disallow all crawlers
  • Specify disallowed paths
  • Sitemap reference support
  • Standard robots.txt format
  • Download ready file
  • 100% free

            
Benefits
A robots.txt file tells search engines which pages to crawl and which to ignore. Essential for protecting private areas and managing crawler traffic.
